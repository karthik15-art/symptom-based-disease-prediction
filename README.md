# 🩺 Disease Symptom Predictor

A Machine Learning-based web application that predicts possible diseases based on symptoms selected by the user. The application uses a **Random Forest** Classifier and provides disease predictions, confidence scores, descriptions, and precautionary measures through an interactive Streamlit interface.

> **Disclaimer:** This project is developed for educational purposes only and should not be considered a substitute for professional medical advice.

---

## 🚀 Features

* Predicts diseases based on user-selected symptoms.
* Interactive and user-friendly Streamlit interface.
* Displays prediction confidence score.
* Provides detailed disease descriptions.
* Suggests precautionary measures.
* Uses Machine Learning for accurate disease classification.

---

## 🛠️ Tech Stack

* **Programming Language:** Python
* **Machine Learning:** Scikit-learn
* **Web Framework:** Streamlit
* **Data Processing:** Pandas, NumPy
* **Model Storage:** Pickle

## ⚙️ How It Works

### 1. Dataset Exploration

* Loads all CSV files.
* Displays dataset shape, columns, and sample records.

### 2. Data Preprocessing

* Cleans symptom values.
* Extracts all unique symptoms.
* Converts symptoms into one-hot encoded format.

### 3. Model Training

* Splits data into training and testing sets.
* Trains a Random Forest Classifier.
* Evaluates model accuracy.
* Saves the trained model.

### 4. Disease Prediction

* Accepts user-selected symptoms.
* Converts symptoms into machine-readable format.
* Predicts the most likely disease.
* Calculates prediction confidence.

### 5. Web Application

* Allows users to select symptoms using a multi-select menu.
* Displays predicted disease, confidence score, description, and precautions.

## 🤖 Machine Learning Model

This project uses the **Random Forest Classifier** because it:

* Handles multi-class classification effectively.
* Reduces overfitting.
* Provides reliable and accurate predictions.

```python
RandomForestClassifier(
    n_estimators=100,
    random_state=42
)
```

## 📦 Installation

### Clone the repository

```bash
git clone https://github.com/your-username/Disease-Symptom-Predictor.git
cd Disease-Symptom-Predictor
```

## 📋 Requirements

Create a `requirements.txt` file containing:

```text
pandas
numpy
scikit-learn
streamlit
```

---

## ▶️ Running the Project

### Step 1: Explore the dataset

```bash
python furi.py
```

### Step 2: Preprocess the dataset

```bash
python clean.py
```

### Step 3: Train the model

```bash
python zoro.py
```

### Step 4: Test predictions

```bash
python predict.py
```

### Step 5: Launch the Streamlit application

```bash
streamlit run lufi.py
```

The application will open in your browser at:

```text
http://localhost:8501
```
